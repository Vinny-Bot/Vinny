Key features and functionalities

Sanction-Based Moderation

Vinny utilizes a sanction-based system for moderation, aiming for consistency and fairness.
Warn - S1 Mute - S2 Tempban - S3 Permban - S4

Open Source

Being free and open source, Vinny allows for community contributions. The code is completely open for anyone to see!

Logging Features

Vinny can log various server activities, including deletions, edits, joins, and leaves.

Origin

Vinny was developed for use in VinegarHQ, though anyone is free to use it!

User-Friendly Dashboard

Vinny features a dashboard for easy configuration and public moderation activity overview.

QuickMod

QuickMod enables quick access to moderation tools (such as warning, muting, and banning) by right clicking a message, hovering over Apps, and clicking on "Quickmod".
